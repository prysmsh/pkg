# github.com/einyx/pkg

Shared Go utilities for Prysm.

## Packages

- **tlsutil** – TLS configuration helpers for post-quantum cryptography
- **pqc** – Post-quantum cryptographic primitives (hybrid KEM: X25519 + Kyber768)

## Usage

```go
import (
    "github.com/einyx/pkg/tlsutil"
    "github.com/einyx/pkg/pqc"
)
```
